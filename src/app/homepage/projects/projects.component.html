Checkers with Minimax AI
This project is a complete implementation of a playable game of checkers against a computer player. The Artificial intelligence uses the Minimax algorithm to determine what moves to make. The heuristics to determine what a good move is and is not was determined by a pseudo genetic algorithm that placed two AI players against one another, one with the previously winning genetic makeup and the opponent with a slightly modified version. If the modified version won more than the previous winner, it becomes the new "best" and the cycle repeats. The project was built using C# and XAML as a WPF Application
My Custom OS
This project is a custom implementation of an operating system using Freestanding C. This project was completed as part of my Fall 2018 Graduate Operating Systems class. The OS is able to read in commands with varying arguments and as a rudimentary login system. Most of the commands that I built into the OS are Arithmetic related however, there are some user related commands as well such as creating a new user and viewing a list of all users.
